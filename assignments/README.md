Mahasiswa wajib mengikuti alur kerja (*workflow*) ini untuk setiap tugas mingguan guna memastikan kode Anda tertata rapi di repositori pusat.

### Langkah 1: Melakukan Fork Repositori

1. Buka halaman repositori utama matkul Anda.
2. Klik tombol **Fork** di pojok kanan atas.
3. Tunggu hingga repositori tersebut muncul di akun GitHub pribadi Anda (misal: `github.com/username-anda/Machine-Learning-Course-2026`).

### Langkah 2: Mengerjakan Tugas di Google Colab

1. Buka Notebook praktikum (misal: `02_Titanic_EDA.ipynb`).


2. Lakukan eksperimen, pembersihan data (*data cleaning*), dan visualisasi sesuai instruksi.


3. Pastikan Anda menyelesaikan tugas tambahan seperti pembuatan fitur `FamilySize` ().



### Langkah 3: Menyimpan ke GitHub Pribadi (Fork)

1. Di Google Colab, klik **File** > **Save a copy in GitHub**.
2. Pilih repositori hasil *fork* Anda.
3. **Penting:** Ubah jalur file (*file path*) agar rapi.
* Format: `assignments/week-X/NIM_NAMA/File_Tugas.ipynb`.
* Contoh: `assignments/week-2/12345_Budi/02_Titanic_EDA_Budi.ipynb`.




4. Klik **OK**.

### Langkah 4: Mengirim Pull Request (Submit)

1. Kembali ke halaman utama repositori **fork** Anda di GitHub.
2. Klik tombol **Contribute** lalu pilih **Open Pull Request**.
3. Beri judul tugas Anda: `Tugas Minggu X - NIM - Nama`.
4. Klik **Create Pull Request**.
